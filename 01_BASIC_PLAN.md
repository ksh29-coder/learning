# Python Adventures for Kids - Basic Course Plan

## Course Overview
**Duration:** 7 days  
**Target Audience:** Children new to programming  
**Language:** Python 3  
**IDE:** Cursor  
**Goal:** Teach core programming concepts through fun, hands-on projects

---

## Day-by-Day Breakdown

### Day 1: Introduction to Programming & Your First Program
**Duration:** 2-3 hours  
**Core Concepts:** What is programming, Python basics, print statements

### Day 2: Variables - The Memory Boxes
**Duration:** 2-3 hours  
**Core Concepts:** Variables, data types, input()

### Day 3: Making Decisions - If/Else Statements
**Duration:** 2-3 hours  
**Core Concepts:** Conditionals, boolean logic, comparison operators

### Day 4: Loops - Doing Things Over and Over
**Duration:** 2-3 hours  
**Core Concepts:** For loops, while loops, range()

### Day 5: Functions - Your Own Commands
**Duration:** 2-3 hours  
**Core Concepts:** Function definition, parameters, return values

### Day 6: Lists - Collections of Things
**Duration:** 2-3 hours  
**Core Concepts:** Lists, indexing, list methods

### Day 7: Putting It All Together - Final Project
**Duration:** 3-4 hours  
**Core Concepts:** Review and integration
