# Day 1 Worksheet - Interactive UI

## How to Run

1. **Open Cursor IDE**
2. **Navigate to the worksheets folder**
3. **Run the UI program:**
   - Right-click on `day1_worksheet_ui.py`
   - Select "Run Python File in Terminal"
   - Or in the terminal, type: `python3 worksheets/day1_worksheet_ui.py`

## Features

### 📝 Interactive Worksheet
- **Part 1: Understanding Print** - Multiple choice and short answer questions
  - **NEW!** Click "✓ Check Answer" button after each question
  - **NEW!** Get instant feedback (✓ Correct or ✗ Try again)
  - **NEW!** See your score update automatically (Score: X/3)
  - **NEW!** Get helpful hints if your answer is wrong
- **Part 2: Code Practice** - Code editor for writing Python code
- **Part 3: ASCII Art Challenge** - Create drawings with code
- **Part 4: Reflection** - Answer reflection questions
- **Bonus Challenge** - Extra coding challenge

### ✅ Answer Checking & Scoring
- **Check your answers** - Click the green "✓ Check Answer" button below each question
- **Instant feedback** - See if you're right or wrong immediately
- **Score tracking** - Watch your score update as you answer correctly (Score: X/3 🌟)
- **Helpful hints** - Get hints if your answer needs improvement
- **Visual feedback** - Green checkmark for correct, red X for incorrect

### 💾 Auto-Save
- Your answers are automatically saved as you type
- Saved to `day1_worksheet_answers.json` in the current directory
- Your work is safe even if you close the program!
- Your checked answers and score are also saved!

### 🧪 Test Your Code
- Click the "🧪 Test My Code" button
- See the output of your code in a popup window
- Great for testing exercises and challenges!

### 📄 View All Answers
- Click "📄 View All Answers" to see everything you've written
- Perfect for reviewing your work

### 🧭 Navigation
- **Scroll down** in Part 1 to see all questions
- **Click tabs** at the top to move between different parts
- All questions are on the same scrollable page in Part 1

## Tips for Kids

1. **Fill in your name and date first** - So you know it's your work!
2. **Work through each tab** - Start with Part 1 and go in order
3. **Check your answers** - Click "✓ Check Answer" after answering each question in Part 1
4. **Watch your score** - Try to get 3/3! 🌟
5. **Read the hints** - If you get something wrong, a helpful hint will appear
6. **Scroll down** - In Part 1, scroll down to see Questions 2 and 3
7. **Test your code often** - Use the "Test My Code" button to see if it works
8. **Don't worry about mistakes** - You can always change your answers and try again
9. **Have fun!** - Be creative with your code and drawings

## Troubleshooting

**Problem:** The window doesn't open
- **Solution:** Make sure Python 3 is installed and tkinter is available
- Try: `python3 -c "import tkinter; print('OK')"`

**Problem:** Can't see the output when testing code
- **Solution:** Make sure you're on the correct tab (Part 2, Part 3, or Bonus)
- Make sure you've written some code first

**Problem:** Answers disappeared
- **Solution:** Check for `day1_worksheet_answers.json` file - your answers should be there!
- The program auto-loads saved answers when you start it

## What Gets Saved?

All your answers are saved in `day1_worksheet_answers.json`:
- Your name and date
- All question answers
- All your code
- Your reflections
- Your notes

You can share this file with your teacher/parent to review your work!
